其实还是有很方便的调试技巧的。

  • 条件断点(Conditional Breakpoints)

    这个调试模式是我最喜欢的，简直不能再方便了，以前遇到在循环里面打断点，需要看某个条件下的值，我只能一遍遍点击，直到满足条件。

#登录

###javaweb中如何去维持登录状态
1.登录后 信息放入 session中
2.页面内验证session中是否有登录信息
3.如果有,不需要再次登录
4.如果没有,跳转登录页面
5.如果登录后点击注销,删除session中登录信息,并清除页面缓存(必要的)

java中对集合排序的方法提供了2种方式。

######这里使用的集合

final List<TestNumModel> list = new ArrayList<>();

  1. 使用的时候传入比较器

    private class Sort implements Comparator<TestNumModel> {
    public Sort() {
    }

    public int compare(TestNumModel tl1, TestNumModel tl2) {
        if (tl1.getNum() < tl2.getNum())
            return -1;
        else if (tl1.getTimePoint() > tl2.getTimePoint())
            return 1;
        else
            return 0;
        }
    }

    使用
