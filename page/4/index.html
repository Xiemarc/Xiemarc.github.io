###PHP 函数参数的2种方式

1）实参的值传给形参，形参的值变化不影响到实参的值，体现了函数的封装性。

2）引用传值，这种方式函数内部的语句会影响到函数外部的变量，破坏了函数的封装性，不推介使用。

    public Sort() {
    }

    public int compare(TestNumModel tl1, TestNumModel tl2) {
        if (tl1.getNum() < tl2.getNum())
            return -1;
        else if (tl1.getTimePoint() > tl2.getTimePoint())
            return 1;
        else
            return 0;
        }
    }
